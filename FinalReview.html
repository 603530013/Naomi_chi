期末考總複習

114 學年度　一年級下學期　國語
範圍：第七課 ～ 第十二課

本卷七大題

  1. 課文內容（12 題）
  2. 注音（8 題）
  3. 部首與筆畫（6 題）
  4. 同字異音（6 題）
  5. 相似字（10 題）
  6. 成語（8 題）
  7. 造句句型（4 題）

共 54 道選擇題，每題點「看答案」可以對照。

一、課文內容（12 題）

1.第七課《借筆》中，小可為什麼會拿大光的筆？
  • A大光送給她
  • B大光不在，她沒問就拿了
  • C大光要她拿
  • D她撿到的
B
2.第七課中，大光後來怎麼回家？
  • A和小可一起
  • B媽媽接他
  • C自己背書包先走了
  • D留在學校
C
3.第八課《兩張卡片》中，他們最後做了什麼？
  • A互相原諒，一起用彩色筆畫畫
  • B還在生氣
  • C分別回家了
  • D找老師告狀
A
4.第八課中，大光忘了帶什麼？
  • A書包
  • B彩色筆
  • C便當
  • D水壺
B
5.第九課《友情的拐杖》中，大山和小彩怎麼幫「我」？
  • A背「我」回家
  • B蹲下來像小矮人，一個在左、一個在右陪「我」走
  • C叫救護車
  • D幫「我」做功課
B
6.第九課中，「我」要去哪裡？
  • A教室
  • B健康中心
  • C家裡
  • D操場
B
7.第十課《一棵小樹》中，故事的「我」是誰？
  • A小朋友
  • B一棵小樹（土裡的種子）
  • C麻雀
  • D小草
B
8.第十課中，小樹最後決定怎麼做？
  • A繼續躲
  • B哭著等媽媽
  • C請小草幫忙
  • D張開雙手，勇敢長大
D
9.第十一課《鉛筆和橡皮擦》中，橡皮擦怎麼幫鉛筆？
  • A擦掉錯字，鼓勵鉛筆加油
  • B幫鉛筆寫字
  • C把鉛筆藏起來
  • D笑鉛筆
A
10.第十一課中，鉛筆說以後還要學什麼？
  • A唱歌
  • B畫畫
  • C跳舞
  • D跑步
B
11.第十二課《上臺說故事》中，他們怎麼解決上臺的問題？
  • A請老師幫忙
  • B不上臺
  • C三人一起上臺，分配角色（千千說故事、亮亮演狼、我演小羊）
  • D用唱的
C
12.第十二課最後，故事說完同學們的反應是？
  • A拍手叫好
  • B沒有反應
  • C笑他們
  • D提早離開
A

七、造句句型（4 題）

1.下列哪一句的句型和「忍不住笑了出來」一樣？
  • A我們笑了出來。
  • B我忍不住哭了出來。
  • C大家很開心。
  • D媽媽笑著說。
B（「忍不住＿了出來」句型）
2.下列哪一句的句型和「一步一步走向健康中心」一樣？
  • A走過小河輕輕的唱。
  • B媽媽笑著說。
  • C一步一步爬上樓梯。
  • D什麼也不說。
C（「一步一步＿」疊字句型）
3.下列哪一句的句型和「又正確又漂亮」一樣？
  • A正確的字。
  • B漂亮極了。
  • C又高又大
  • D真的好漂亮。
C（「又＿又＿」句型）
4.下列哪一句的句型和「我做到了，我們做到了！」一樣？
  • A大家都做完了。
  • B我成功了，我們成功了！
  • C故事說完了。
  • D同學們拍手叫好。
B（個體＋集體疊字強調的句型）
